인터페이스

다른 클래스에서 같은 기능을 사용하고 싶을 때

목적: 벌들이 2개이상의 역할을 수행할 수 있도록 하는것

특정 클래스에 인터페이스를 등록하면 인터페이스가 가지고 있는 메소드를 구현해야함

아니면 컴파일 오류 발생

필드는 정의하지 않고 속성을 사용 -> 자동으로 Public으로 지정

특징:

1) 인터페이스 interface = new 인터페이스(); 안돼요

2) 인터페이스 interface = 객체; 이렇게 참조변수로 활용하는건 가능!!!
인터페이스를 구현하는 클래스라면 2)의 상황을 활용할 수 있음

3) if(객체 is interface) 어떤 클래스의 객체가 interface를 구현한다면이라는 뜻
